# Github-commands


git clone <url>
git checkout <remote-branch>
git checkout -b <to-ur-new-branch>
git fetch, brings the remote changes of the current branch, doesnt apply the changes
git status, checks ur local with the fetched and mentions are you up  to date with remote branch
git pull, brings the remote changes of the current branch, it does apply the changes on ur local
git revert --no-commit <commitHash>
